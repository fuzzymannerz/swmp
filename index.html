SWMP

Server Web Monitor Page

A responsive, eye-pleasing Linux server statistics dashboard.
wget https://get.swmp.ml/install.sh && chmod +x install.sh && ./install.sh


Features:

  • System Info
  • RAM & Swap Usage
  • CPU Average Load
  • Network & Storage Info

Plus:

  • Responsive Design
  • A Choice Of Themes
  • Lightweight (~300kb)*
  • Completely Free!

    • * Without the extra themes.

Requirements:

  • Linux OS
  • Web Server
  • PHP
