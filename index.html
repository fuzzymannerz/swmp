SWMP

Server Web Monitor Page

A responsive, eye-pleasing Linux server statistics dashboard.
wget https://raw.githubusercontent.com/fuzzymannerz/swmp/gh-pages/install.sh && chmod +x install.sh && ./install.sh

Features:

  • System Info
  • RAM & Swap Usage
  • CPU Average Load
  • Network & Storage Info

Plus:

  • A Choice Of Themes
  • Various Settings
  • Language Selection
  • Lightweight (<1MB)*
  • Completely Free!

    • * Without the extra themes.

Requirements:

  • Linux OS
  • Web Server
  • PHP
